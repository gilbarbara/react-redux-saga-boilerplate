### Building
- webpack ^3.x

`npm run build`

### Development
- webpack-dev-server ^2.x
- react-hot-loader ^3.x

`npm start`

### Tests
- jest ^21.x
- enzyme ^3.x

`npm test`  
`npm run test:watch`

### Browser Automation
- nightwatch ^0.9
- selenium ^3.0

`npm run test:ui` (with dev-server already running)  
`npm run test:ui:start` (start dev-server, run tests and exit)
